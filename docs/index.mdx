# What is Bauxite?

Put simply, Bauxite is a suite of tools for managing Minecraft servers, including:

- [Command-line tools](/cli)
- [Node.js APIs](/api)

These tools can be used for:

- Managing local Minecraft servers

Many, many more features are planned for the future. For more details, see [the roadmap](/roadmap).
